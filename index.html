2. The Classic Float Technique (Three-Box Layout)

Before Flexbox and Grid, **Floating** was the primary way to achieve multi-column layouts. It moves the box to the side, allowing subsequent content to wrap around it. We use the **clearfix** pattern to contain the floated elements.

CSS Techniques Used

  • **Box Model:** `box-sizing: border-box`
  • **Flexbox:** `justify-content`, `align-items`
  • **Grid:** `grid-template-columns`, `grid-gap`
  • **Float:** `float: left`, **Clearfix**
  • **Positioning:** `position: sticky`
  • **Functions:** `var()`, `calc()` (implicitly used by `fr` units)
  • **Transitions:** `transition: all 0.3s`
  • **Media Queries:** Responsiveness
